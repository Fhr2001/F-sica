Programa criado por Fábio Henrique e produzido utilizando a linguagem de programação Python 3.0

Este gera uma matriz de tamanho 50x3, onde as colunas estão referidas as dimensões vetoriais X, Y e Z.
Na matriz é aplicada uma função randomizadora de numeros variando entre -10 e 10.


Nele é possivel tomar 7 decisões, as quais são orientadas no próprio programa, caso tenha interesse em ler por aqui são respectivamente:

1: imprimir a matriz completa.
2: Re-gerar a matriz, podendo inclusive alterar a amplitude de randomização.
3: Imprimir uma linha expecifica 
4: Alterar uma linha em específico manualmente
5: Calcular Força Resultante
6: Calcular a Aceleração do objeto
